# Lingua — Frontend

React · TypeScript · Vite · Tailwind CSS

## Çalıştırma

```bash
cd frontend
npm install
npm run dev
```

Uygulama http://localhost:5173 adresinde çalışır. Backend API adresi `frontend/.env` dosyasındaki `VITE_API_URL` değişkeniyle yapılandırılır (varsayılan: `http://127.0.0.1:8010`).
